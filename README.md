This application features a QML-based user interface and a Python backend. 
The backend includes 'bridge.py' for communication, 'main.py' to run the app, 
and 'logic.py' which handles the logic for preparing file paths 
for open and save operations, 
ensuring compatibility across Linux and Windows.
